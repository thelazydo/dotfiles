# macOS Keyboard Shortcuts (Formatted as Markdown Tables)

---

## Modifier Keys

| Key                    | Description                                           |
| ---------------------- | ----------------------------------------------------- |
| Fn (Function) or Globe | Function key (used for F1-F12 and special actions)    |
| Control (Ctrl) ⌃       | Modifier key, often used for system-level commands    |
| Option (Alt) ⌥         | Modifier key, similar to Alt on Windows               |
| Shift ⇧                | Modifier key for capitalization and alternate actions |
| Command (Cmd) ⌘        | Primary shortcut modifier on macOS                    |
